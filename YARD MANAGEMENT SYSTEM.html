📖 How Drivers Use This
1. Dispatcher uploads Yard List (Tab 1) and LoadSumm (Tab 2).
2. Share this page URL with drivers via WhatsApp/text.
3. Driver opens on phone browser → clicks Driver View tab.
4. Driver types their Booking # → sees their next container, exact location, seal # and weight.
5. Dispatcher marks containers as Assigned ✓ in Tab 2 — those disappear from driver view automatically.

💡 To make this a live website: Host this HTML file on a free service like Netlify Drop (drag-and-drop), GitHub Pages, or ask your IT team to put it on your company server. Drivers then open the URL on any phone.
